# Contributing to AI Moderated Encyclopedia

Thank you for your interest in contributing to the AI Moderated Encyclopedia! Your contributions help improve the project and ensure it remains a valuable, community-driven resource. Please take a moment to review the guidelines below.

## How to Contribute

### 1. Reporting Issues
If you encounter a bug, have a feature request, or notice something that needs improvement:
- **Open an Issue:** Provide a clear and descriptive title.
- **Describe the Problem:** Include steps to reproduce the issue (if applicable) and any relevant details.
- **Feature Requests:** Explain why you believe a new feature would be beneficial.

### 2. Submitting Pull Requests
We welcome pull requests (PRs) for bug fixes, new features, and improvements. Before submitting a PR, please:
- **Fork the Repository:** Create your own copy on GitHub.
- **Create a New Branch:** Use a descriptive name, for example:  
  ```bash
  git checkout -b feature/your-feature-name
  ```
- **Make Your Changes:** Follow the existing code style (see below), add tests if necessary, and ensure your code is well-documented.
- **Commit Your Changes:** Write clear and concise commit messages.
- **Push to Your Fork:**  
  ```bash
  git push origin feature/your-feature-name
  ```
- **Open a Pull Request:** Submit your PR against the main repositoryâ€™s default branch. A project maintainer will review your changes.

### 3. Code Style and Best Practices
- **Style Guide:** Follow [PEP 8](https://www.python.org/dev/peps/pep-0008/) for Python code.
- **Documentation:** Include comments and documentation where necessary.
- **Testing:** Ensure your changes do not break existing functionality and add tests when adding new features.

### 4. Communication and Feedback
- **Discussion:** Use GitHub issues or pull request comments to discuss ideas, ask questions, and provide feedback.
- **Respect Decisions:** While all contributions are welcome, please note that only project maintainers have the authority to merge changes. Merging decisions are made to maintain the integrity and direction of the project.

### 5. Licensing
By contributing, you agree that your contributions will be licensed under the project's chosen license (GPL v3).

Thank you for helping make the AI Moderated Encyclopedia better for everyone!
